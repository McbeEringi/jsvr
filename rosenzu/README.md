# 路線図かくやつ

data.txtに更新あったら適当にプルリク出してくれると嬉しいです

## Requirements
- node
- imagemagick
- make

Windowsでも動くがimagemagickが全然だめなのでブラウザ等で手動でpngにする必要あり

プルリクください

## Usage
### pngが欲しい
#### Linux
1. data.txtに書く
1. `make`
1. out.pngを確認

#### Windows
1. svgを得る cf. svgが欲しい
1. out.svgを適当に変換する

### svgが欲しい
1. data.txtに書く
1. `make svg`
1. out.svgを確認
